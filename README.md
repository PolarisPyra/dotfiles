# Dotfiles

This repository contains my dotfiles configuration. Below is a list of programs needed to use these dotfiles.

## Shell & Prompt
- **zsh** (main shell)
- **oh-my-zsh** (zsh framework)
- **starship** (prompt)

## Terminal Emulators
- **wezterm**

## File Manager
- **[Yazi](https://yazi-rs.github.io/)**
- **[Clipboard](https://github.com/Slackadays/ClipBoard)**

## Editor
- **[nvim](https://neovim.io/)** (Neovim)

## Menu/Launcher
- **[fuzzel](https://codeberg.org/dnkl/fuzzel)** (used in scripts for dmenu-like prompts)

## Music & Audio
- **mpd** (Music Player Daemon)
- **[rmpc](https://github.com/mierak/rmpc)** (MPD client)

## Window/Bar/Compositor
- **hyprland** (hypr config present)
- **waybar** (waybar config present)
- **swaync** (notification daemon)
- **waypaper** (wallpaper setter)
- **wlogout** (logout menu)

## Node & Package Management
- **nvm** (Node Version Manager)
- **pnpm** (Node package manager)
